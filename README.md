# HackMentor-AI# 🚀 HackMentor AI

> Transform your ideas into winning hackathon pitches — powered by AI

---

## 🧠 Overview

HackMentor AI is an intelligent, AI-powered platform designed to help beginner and non-technical students confidently participate in hackathons.

It acts as a **virtual hackathon mentor**, guiding users step-by-step from idea generation to a complete, structured, and presentation-ready solution.

---

## ❗ Problem

Many students struggle in hackathons due to:
- Lack of clarity in choosing ideas
- Confusion about tech stack
- Difficulty structuring solutions
- Fear of presenting in front of judges

This creates a barrier for:
- 1st/2nd year students
- Non-technical backgrounds (Mechanical, Civil, etc.)

---

## 💡 Solution

HackMentor AI bridges this gap by converting simple inputs (ideas/domains) into:

- Structured problem statements  
- Innovative solutions  
- Clear technical stacks  
- Complete pitch content  
- 60-second presentation scripts  
- AI-powered evaluation and feedback  

---

## 🔥 Features

### 🧠 Idea Generator
Generate multiple hackathon-ready ideas from a simple domain.

### ⚙️ Solution Builder
Get structured features, architecture, and easy-to-understand tech stack.

### 🎤 Pitch Generator
Create a complete hackathon pitch including:
- Problem
- Solution
- USP
- Impact
- 60-sec script

### 📊 AI Scoring System (USP)
Evaluate your idea based on:
- Innovation  
- Feasibility  
- Impact  

With improvement suggestions.

---

## 💣 Unique Value Proposition

Unlike general AI tools like ChatGPT, HackMentor AI:

- Provides **guided workflows**, not just responses  
- Focuses specifically on **hackathon success**  
- Evaluates and improves ideas  
- Designed for **non-technical users**

## 🛠️ Tech Stack

- **Backend:** FastAPI (Python)
- **AI Engine:** OpenAI API / LLM
- **Frontend:** HTML, CSS, JavaScript (React optional)
- **PPT Generation:** python-pptx
- **Data Handling:** JSON / SQLite (future)

---

## 🚀 How It Works

1. Enter a domain or idea  
2. Get multiple AI-generated ideas  
3. Select and refine one  
4. Generate solution and tech stack  
5. Create full pitch  
6. Receive AI-based score and feedback
